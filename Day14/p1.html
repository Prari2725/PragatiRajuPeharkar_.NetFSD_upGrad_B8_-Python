FAQ Section

1. What is jQuery?
jQuery is a JavaScript library used for DOM manipulation.
2. Why use jQuery?
It simplifies JavaScript coding and UI interactions.
3. What is slideToggle()?
slideToggle() shows or hides elements with sliding animation.
4. Can jQuery manipulate HTML?
Yes, it can add, remove, and modify HTML elements.
5. Is jQuery still used?
Yes, many legacy and UI projects still use it.
